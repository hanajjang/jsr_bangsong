# 곡명 카테고리 비고
⏳ 곡목 데이터 로드 중...
이슈 로그
💡
물리적 확인 우선 원칙
장비 이슈 발생 시 — ① 케이블·연결부 확인 → ② 전원 재시작 → ③ 설정 확인 → ④ 담당자 연락 순으로 진행하세요.
방송 시스템 가이드
✅ 예배 전 체크리스트
⌄
  • 1방송 PC 전원 ON → 모니터 신호 확인
  • 2빔프로젝터 전원 ON → 워밍업 5분 대기
  • 3음향 믹서 전원 ON → 앰프는 마지막에
  • 4ProPresenter 실행 → 해당 주일 파일 로드
  • 5무선마이크 배터리 잔량 확인 (2칸 이상)
  • 6화면 출력 테스트 — 슬라이드 정상 출력 확인
  • 7음향 레벨 체크 — 찬양팀과 사운드체크 진행
  • 8카메라 앵글 및 포커스 최종 확인
🔌 장비 전원 ON 순서
⌄
1방송 PC & 모니터
↓
2빔프로젝터 (워밍업 시작)
↓
3음향 믹서 & 이펙터
↓
4무선마이크 수신기
↓
5파워 앰프 (항상 마지막)
⚠️ 앰프는 반드시 마지막에 켜고 가장 먼저 꺼야 합니다.
🔴 장비 전원 OFF 순서
⌄
1파워 앰프 (항상 먼저)
↓
2무선마이크 수신기
↓
3음향 믹서 & 이펙터
↓
4빔프로젝터 (쿨다운 완료 후)
↓
5방송 PC 종료
🧹 예배 후 정리 체크리스트
⌄
  • 1무선마이크 전원 OFF → 배터리 분리 보관
  • 2마이크 스탠드 정위치 보관
  • 3케이블류 정리 및 원위치
  • 4ProPresenter 파일 저장 후 종료
  • 5이슈 발생 시 이슈 로그에 즉시 기록
  • 6방송실 전등 끄고 문 잠금 확인
🖥️ ProPresenter 기본 사용법
⌄
  • 1라이브러리에서 해당 날짜 파일 더블클릭 → 로드
  • 2화면 출력: 슬라이드 클릭 → 엔터 or 스페이스바
  • 3검정화면(Blank): 단축키 B
  • 4로고화면(Logo): 단축키 L
  • 5슬라이드 번호 틀릴 때 → 재시작 (⌘R / Ctrl+R)
  • 6화면 먹통 시 → Stage Display 설정 확인
